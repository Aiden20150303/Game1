# 게임 규칙

> 이 문서는 '장애물 피하기' 게임(Body Dodge)의 규칙을 정의합니다.

## 🎮 게임 제목

**Body Dodge (바디 닷지)**

부제: 온몸으로 장애물을 피하라!

---

## 📝 게임 설명

화면 위쪽에서 떨어지는 장애물(똥/돌멩이)을 피하고, 
보너스 코인을 획득하여 최고 점수에 도전하는 게임입니다.

플레이어는 몸을 왼쪽, 오른쪽으로 기울이거나 정면을 유지하여 
캐릭터를 3개의 라인(Left, Center, Right) 사이로 이동시킬 수 있습니다.

---

## 🎯 게임 목표

- 날아오는 **장애물(폭탄)을 피해서** 최대한 오래 생존하기
- **코인**을 획득하여 점수 높이기
- 목표: 1000점 달성!

---

## 🕹️ 게임 규칙

### 기본 규칙

1. **시작**: "Start" 버튼을 누르면 3초 카운트다운 후 시작
2. **이동**: 
   - 플레이어는 화면 하단 중앙에 위치
   - 포즈(Left/Center/Right)에 따라 즉시 해당 라인으로 이동
3. **충돌 판정**:
   - 💣 **폭탄**: 닿으면 **HP 1 감소** (총 3개)
   - 💰 **코인**: 닿으면 **점수 +100점**
4. **종료 조건**:
   - HP가 0이 되면 게임 오버
   - (선택사항) 제한 시간 없음 (오래 버티기)

### 레벨 시스템

- 시간이 지날수록 장애물이 떨어지는 속도가 빨라짐
- **Level 1**: 기본 속도
- **Level 2** (30초 후): 속도 1.2배
- **Level 3** (60초 후): 속도 1.5배 + 폭탄 등장 빈도 증가

## 🎨 필요한 포즈 목록 (클래스)

Teachable Machine에서 학습시켜야 할 포즈입니다.

| 포즈 이름 (Label) | 설명 | 게임 동작 |
|------------------|------|----------|
| **Left** | 고개를 왼쪽으로 갸웃하거나 몸을 왼쪽으로 기울임 | 캐릭터가 **왼쪽 라인**으로 이동 |
| **Center** | 정면을 바라보고 바르게 선 자세 | 캐릭터가 **가운데 라인**으로 이동 |
| **Right** | 고개를 오른쪽으로 갸웃하거나 몸을 오른쪽으로 기울임 | 캐릭터가 **오른쪽 라인**으로 이동 |

> **Tip**: "Center"는 아무것도 하지 않는 기본 상태로 학습시키면 편합니다.

## 💡 개발 가이드 (AI 참고용)

이 규칙을 바탕으로 코드를 수정할 때 참고할 사항:

1. **라인 시스템**: 화면 가로를 3등분하여 좌표 설정 (33%, 50%, 66% 지점)
2. **낙하 로직**: 아이템은 랜덤한 라인(Left/Center/Right)에서 생성되어 아래로 떨어짐
3. **충돌 처리**: 플레이어와 아이템의 거리 계산 
4. **상태 관리**:
   - `gameState`: READY, PLAYING, GAMEOVER
   - `playerPos`: 0 (Left), 1 (Center), 2 (Right)
